Gem for tracking bitcoin markets. Supports MtGox at the moment.

## Installation

Add this line to your application's Gemfile:

    gem "bitbot-live", github: "indrekj/bitbot-live"

And then execute:

    $ bundle

## Usage

See scripts/trades.rb and scripts/ticker.rb
